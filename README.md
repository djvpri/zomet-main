# Zomet - Ekosistem Aplikasi Digital

Landing page terpusat untuk semua aplikasi dalam ekosistem Zomet.

## 🚀 Aplikasi

- **ZSnap** - AI Screenshot Solver
- **ZGold** - Jewelry Store POS
- **ZBengkel** - Workshop POS
- **ZLaundry** - Laundry POS
- **ZPos** - Digital Cashier
- **ZFace** - Face Recognition System
- **ZGym** - Gym Management
- **ZOne** - Admin Hub

## 📋 Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## 🛠️ Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 📦 Build

```bash
npm run build
npm start
```
